## よく使うコマンド早見表

| コマンド | 説明 |
|----------|------|
| `claude` | Claude Code を起動 |
| `claude --continue` | 前回の会話を続ける |
| `claude --resume` | 過去の会話を選んで再開 |
| `/help` | ヘルプ・スキル一覧を表示 |
| `/exit` | 会話を終了 |
| `/clear` | 会話履歴をクリア |
| `/compact` | 会話を要約して圧縮 |

!!! tip "ブックマークのすすめ"
    よく参照するリンクはブラウザのブックマークに保存しておくと便利です。
