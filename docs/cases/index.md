# 事例集

Claude Code を実際の業務で使う **具体的な事例** をまとめたページです。

「Claude Code が何をしてくれるのか」は概念だけだとイメージしづらいので、**実在のサービス・実在の作業フロー** に当てはめた手順を順番にたどれる形で紹介します。

## 想定読者

- Claude Code の基本操作（Part 1〜3）は完了している方
- 自分の仕事に Claude Code をどう組み込めばいいか **具体例が欲しい** 方
- IT の専門知識（SSH・Docker・Git など）は **詳しくない** が、手順通りに進めれば作業できる方

専門用語は出てくるたびに簡単な補足を入れています。コマンドはコピーして貼り付ければ動く形で書いています。
