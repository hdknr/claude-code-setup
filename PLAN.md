# Bootstrp Claude

## 目的

- 知人が新品の MacBook を購入しました
- Mac Bookは 稼働すみで iCludeも設定されています
- 彼の Mac Bookで Claude Code をインストールして使える様にするための手順をを作ってください

## 利用者の前提

- Githubについてくわしくありません
- システム開発者ではないのでLinxコマンドなどは知りません
- 利用用途は主に彼の日々のMac Bookを使った活動支援です(提案書,計画書、写真などクリエイティブ制作物のポートフォリオ公開、履歴書作成、調査過程の思考の管理, 確定申告など政府系への提出書類の整理、など)
- 私に質問がオフラインでくるので私が説明しやすい様な環境で構築したい

## 環境

- githubを使う
- ローカルツールは homebrew を使う
- テキストエディタは VSCodeを前提とはしない

## 手順書の内容

- 3パートにわける。 1.事前準備, 2. Claude Code インストール (アカウント登録からの手順) 3. インストール後の環境準備
- Cluade Codeをインストールしてから指示で実行可能なことは、すべて 3で実装します
- 3に関してはなるだけプロンプト入力させたくないので、事前にスキルを作成してドキュメント化する
- そのために、ここでの成果物は githhub.com/hdnr/bootstrap-claude (仮称)に公開レポジトリを用意して、ドキュメントとスキルなどを一般に再利用できる様にする
